## How-to edit chti.octo.com

Jekyll frameworks enabled you to edit data yaml files
 ### Add new people :
- Edit [people.yml](_data/people.yml)
- copy-paste this template :
```
- poly:
  name:
  job:
  accroche: |
    ...
  bio: |
    ...
```
### Add new bbl :
- Edit [bbl.yml](_data/bbl.yml)
- copy-paste this template :
```
- title: ""
  poly:
  description: |
    ...
```

### Add new job :
- Edit [jobs.yml](_data/jobs.yml)
- copy-paste this template :
```
- title:
    description: |
        ...
    url:
    image:
    datePosted:

```
